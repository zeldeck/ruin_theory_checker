# Симулятор теории разорения

Скрипт, моделирующий игру в монетку. Хотя кажется что шансы равны 50/50, чем дольше играешь, тем больше шансов проиграть всё.
Игра такая: кидается монетка, если выпала 1, то и вы получаете +1, если 0 то -1. Итак пока у вас не будет на счету ноль.

## Что реализовано в этом проекте:
* **Старт назначает игрок:** Поскольку тут важен факт, а не количество. То вы можете сами выбрать сколько монет на старте.
* **Счетчик раундов:** программа считает точное количество шагов до обнуления.
* **Изолированная среда:** проект настроен внутри виртуального окружения.

## Стек технологий:
* Python 3.13
* Встроенная библиотека `random`
* Системный контроль версий `Git / GitHub`

## Как запустить проект:
1. Склонировать репозиторий к себе на компьютер.
2. Активировать виртуальное окружение `.venv`.
3. Запустить главный файл:`ruin_coin.py`
